# Abnormal Detection
This is part of my work during IEEE SMC on trying to use EEG for concussion detection, aimed to detect concussion for non-specialists by simply putting on the EEG. 
To achieve this goal, I used the abnormal data from TUAB dataset. The test accuracy of the best model achieved 96.87% detection accuracy. However, I only train and 
test the model on a small subset of the TUAB dataset due to computational limitations, which is around 1GB of dataset.
